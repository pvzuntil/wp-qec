== Description ==

Eduline is a simple, clean, responsive and user-friendly education WordPress theme specially designed for university, school, college or other educational institutions. The Eduline theme has powerful theme customization options such as custom header settings, logo, custom menu, social icons, featured slider, courses and more. It makes your website more engaging and informative.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
